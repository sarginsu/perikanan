# Cara menjalankan Dahboard Efishery

#### Langkah-langkah untuk manjalankan Aplikasi
```sh
git clone https://github.com/sarginsu/perikanan
cd perikanan
npm i
npm start
```

#### Buka browser di alamat ini
```
localhost:3000
```
